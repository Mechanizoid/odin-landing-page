# Odin Landing Page

## Background
This is an exercise in the Foundations course at [The Odin Project](https;//www.theodinproject.com/). Specifically, this project is the final assignment in the Flexbox section, and the assignment is to create and style an entire web page to match the design provided by TOP. Full details of the assignment may be found at [Project: Landing Page](https://www.theodinproject.com/lessons/foundations-landing-page).

## Goals
The goal of this project is to apply the basic knowledge of HTML and CSS that I acquired in the Foundation course. By completing this assignment, i demonstrate that I can use what I've learned so far to create a layout from scratch.
